## 기능 개선 (스펙 Gap)

### CHANGELOG 랜딩 뷰어
- [ ] 대시보드에 CHANGELOG 전용 페이지/탭 추가
- [ ] 프로젝트별 CHANGELOG 히스토리 조회
- [ ] 버전별 변경사항 필터링/검색
- 현재: 태스크별 `.claude/tasks/TASK-XXX/CHANGELOG.md`에만 저장

### 자동 롤백
- [ ] BACKLOG 전환 시 git stash 또는 branch reset
- [ ] 롤백 전 확인 다이얼로그
- [ ] 롤백 이력 기록
- 스펙 요청사항: "완료에 도달할 만한 사항이 아니면 backlog로 보내 변경사항은 롤백"

### 자동 서버 실행 연동
- [ ] 프론트엔드 태스크 완료 시 자동으로 dev server 실행
- [ ] TaskModal에서 서버 URL 바로가기 제공
- [ ] 검증 방법 안내 (테스트 명령어 등) 자동 생성
- 스펙 요청사항: "사용자가 해당 내용을 확인할 수 있도록 방법을 제시"

### 작업 의존성
- [ ] 태스크 간 의존성 설정 (depends_on)
- [ ] 선행 태스크 완료 시 자동 READY 전환
- [ ] 의존성 그래프 시각화
